svn常用功能及插件使用

Windows平台上创建SVN服务，方便启动和关闭；Eclipse中svn提交时忽略文件；

创建svn服务

开启：net start svn
关闭：net stop svn

Eclipse中svn提交时忽略文件

Windows –> preferences –> Team –> Ignore Resources –> Add Pattern
之后分别添加下面四项，并点击OK

.classpath
.project
target
.settings
.externalToolBuilders

subeclipse切换账户

C:\Users\Administrator\AppData\Roaming\Subversion\auth

删除其中所有文件
